# 安全审查技能

## 提交前安全检查清单

### 必查项目
- [ ] 无硬编码密钥（API 密钥、密码、令牌）
- [ ] 所有用户输入已验证
- [ ] 参数化查询（防 SQL 注入）
- [ ] HTML 输出已转义（防 XSS）
- [ ] CSRF 保护已启用
- [ ] 认证和授权已验证
- [ ] 端点有速率限制
- [ ] 错误消息不暴露敏感信息

### 密钥管理
- 使用环境变量或密钥管理服务
- 启动时验证密钥可用性
- 泄露的凭证立即替换
- 绝不在代码中嵌入密钥

### API 安全
- 所有端点需要认证（除公开接口）
- 基于角色的访问控制
- 请求体大小限制
- 输入格式验证

### 数据安全
- 敏感数据加密存储
- 传输使用 HTTPS
- 日志中脱敏处理
- 定期轮换密钥

## 安全事件响应

1. 停止当前工作
2. 评估漏洞严重性
3. 修复关键漏洞
4. 替换泄露的凭证
5. 全面审计类似问题
